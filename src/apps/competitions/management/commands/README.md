# 提交记录时间修复脚本

## 功能说明

`fix_submission_dates.py` 是一个Django管理命令，用于修复提交记录的时间问题。由于系统之前使用的是美国时间而非北京时间（CST），导致部分提交记录的时间不正确。

这个命令会：
1. 查找所有在2025年4月18日12:00之后创建的提交记录
2. 为这些记录的创建时间（created_when）添加8小时，将美国时间转换为北京时间（CST）

## 使用方法

### 运行命令

```bash
# 在Django容器中运行
docker compose exec django python manage.py fix_submission_dates
```

运行后，脚本会：
1. 显示找到的需要修复的记录数量
2. 询问是否继续执行修复操作
3. 如果确认，将为所有匹配的记录添加8小时
4. 显示修复完成的记录数量

### 注意事项

- 此脚本会直接修改数据库中的记录，请在执行前确保已备份数据库
- 脚本执行过程中会要求确认，输入 'y' 确认继续
- 如果记录数量较多，脚本执行可能需要一些时间
